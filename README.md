# HenoriaBot

Le HenoriaBot est fait pour pouvoir gérer le serveur SkyBlock "Henoria"

# Installation

Pour installer tous les package nécessaires :

```js
npm install
```

Ensuite pour le lancer vous pouvez executer ces deux commandes

```js
npm run start
```
```js
node src/main.js
```
